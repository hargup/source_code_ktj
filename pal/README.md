The task in this problem statement is to find the length of the longest palindrome in a string given as input.

Input : 
Enter a non-spaced string.
Constraints :
length of string < 10^6

Output : 
Length of the longest palindromic substring 

Example :

abcdef
1
abcabc
a
abba
4
abbaa
4
abadeed
4
jhkkhjmnm
6
